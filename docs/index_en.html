Features

Key features

Built for Windows, with mobile access and extensibility in mind.

AI
LLM tag auto-generation
Supports Gemini API, LM Studio, and any OpenAI-compatible API. Tags generated naturally from character names.
L4

Setup

Up and running in 5 minutes

On Windows, just double-click start_anima_pipeline.bat.

1
Export workflow JSON from ComfyUI
Browse Templates → Anima → Save (API Format), then place the file in workflows/.
2
Run the batch file
Double-click start_anima_pipeline.bat. Dependencies are installed automatically on first run.
3
Open localhost:7860
Open the UI in your browser, set the ComfyUI URL and LLM settings, then save.
4
Enter a character and generate
Press Ctrl+Enter to generate. Results appear in the UI immediately.
