# v1.5.1 Release Notes / リリースノート
**Release Date / リリース日:** 2026-03-24

---

## Overview / 概要
> v1.5.1 is a documentation and release-governance update that formalizes post-refactor API references and repository hygiene for stable maintenance.

> v1.5.1 は、リファクタ後のAPI参照とリポジトリ運用ルールを正式化し、保守運用を安定化するためのドキュメント/運用整備リリースです。
