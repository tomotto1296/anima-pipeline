## Goal / 目的

毎回設定を考えるのが面倒なとき、ランダムなテンプレート（キャラ＋シーン＋品質タグのセット）を一発で適用して即生成できるようにする。

---

## Scope

- **In Scope:**
  - ボタン1クリックでキャラ・シーン設定をランダムに埋める
  - キャラ: 既存プリセットからランダムに1件ロード（プリセットがある場合）
  - シーン: `settings/ui_options.json` の既存データを流用してランダム選択
  - 適用後は通常通り生成ボタンで生成できる

- **Out of Scope:**
  - テンプレートのカスタマイズ・保存
  - シード値のランダム設定（既存のランダムシード機能で対応済み）
  - 属性をランダムに組み合わせた新規キャラ生成（INPUT-1の役割）

---

## User Story

> As a ユーザー, I want 今日の気分ボタン1クリックで設定が埋まる, so that 何を作るか決まっていないときでもすぐ生成を始められる.

---

## Requirements / 要件

1. 生成ボタン付近に「🎲 今日の気分」ボタンを追加する。
2. クリックで以下をランダムに設定する:
   - キャラ: 既存キャラプリセットからランダムに1件ロード（`GET /chara_presets` を使用。プリセット0件の場合はスキップ）
   - シーン世界観: `ui_options.json` の `scene_world` からランダム選択
   - 時間帯: `ui_options.json` の `scene_tod` からランダム選択
   - 天気: `ui_options.json` の `scene_weather` からランダム選択
   - 品質タグ: 現在の設定を維持（変更しない）
3. 設定適用後に「今日の気分: <適用内容のサマリー>」をSTATUSに短時間表示する。
4. 適用後はユーザーが手動で変更できる（上書き可能）。
