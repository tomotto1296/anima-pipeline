Anima Pipeline Demo

キャラクター生成（プロンプト+画像） 体験版

注意: GitHub Pages (HTTPS) 上からローカル ComfyUI（http://127.0.0.1:8188）へは、ブラウザのセキュリティ制限で接続できない場合があります。
ただしローカルで ComfyUI が起動していれば生成できます。
これは何？
このページは「Anima Pipeline」の簡易体験版です。
  • できること: キャラ条件を選んで画像生成、キャラプリセット保存（JSON保存）
  • できないこと: 本番の全機能（詳細設定・高度な編集・一部連携機能）
  • 使い方: 1) ComfyUI URLを確認 2) キャラ条件を選択 3) 生成ボタンを押す

ワークフローは固定 (`image_anima_preview.json`) です。JSONカスタマイズは不可にしています。

公開版ページ（GitHub Pages / HTTPS）から `http://127.0.0.1:8188` への接続は、ブラウザ制約で失敗する場合があります。ローカルでページを開く（例: `http://localhost:7860`）か、HTTPS 経由で公開した ComfyUI を利用してください。

生成パラメータ

保存するとJSONファイルをダウンロードします（保存先はブラウザの既定フォルダ）。
